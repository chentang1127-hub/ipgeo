IP to Country API —
Know Where Your Users Are

The simplest way to get a country code from an IP address. 99.8% accuracy, sub-millisecond lookups, 200+ countries covered. Free for 10,000 queries/month.

One Request, One Country

Send any IPv4 or IPv6 address — get back the ISO country code, full name, and continent:

curl -H "X-API-Key: ipgeo_YOUR_KEY" \
  https://api.getipgeo.com/v1/ip/8.8.8.8

Response:

{
  "ip": "8.8.8.8",
  "country": {
    "code": "US",
    "name": "United States"
  },
  "continent": {
    "code": "NA",
    "name": "North America"
  },
  "timezone": "America/Chicago"
}

curl -H "X-API-Key: ipgeo_YOUR_KEY" \
  https://api.getipgeo.com/v1/ip/8.8.8.8 \
  | jq '.country.code'
import requests

resp = requests.get(
    "https://api.getipgeo.com/v1/ip/8.8.8.8",
    headers={"X-API-Key": "ipgeo_YOUR_KEY"}
)
data = resp.json()
print(data["country"]["code"])  # "US"
const resp = await fetch("https://api.getipgeo.com/v1/ip/8.8.8.8", {
  headers: { "X-API-Key": "ipgeo_YOUR_KEY" }
});
const data = await resp.json();
console.log(data.country.code); // "US"
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, "https://api.getipgeo.com/v1/ip/8.8.8.8");
curl_setopt($ch, CURLOPT_HTTPHEADER, ["X-API-Key: ipgeo_YOUR_KEY"]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
$data = json_decode(curl_exec($ch), true);
echo $data["country"]["code"]; // "US"
