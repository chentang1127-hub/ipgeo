Changelog

Every update to the IPGeo API, documented.

June 15, 2026

Feature Risk Detection

Added risk_flags field to all lookup responses. Detects Tor exit nodes via the public Tor exit list (refreshed hourly) and hosting/datacenter IPs via 21 known provider ASNs. Zero-cost, zero-external-API-dependency. When no risk signals are detected, the field is omitted entirely.

June 15, 2026

Feature Free Tools: IP Lookup, My IP, ASN Lookup

Launched three free web-based tools — IP geolocation search, automatic IP detection, and ASN/ISP lookup. All powered by the same MaxMind data as the API. No signup required for the web tools.

June 14, 2026

Feature Public Stats & Changelog

Added /v1/stats public endpoint and /public-stats page showing total lookups served, uptime, and cache performance. Launched this changelog for transparent product updates.

June 14, 2026

Improve Positioning & Landing Page

Rewrote home page hero to focus on "Fortune-500 accuracy, startup-friendly prices." Replaced weak social proof with concrete trust signals: MaxMind GeoIP2, OpenAPI 3.1, Cloudflare CDN. Added comparison pages for MaxMind and IPinfo.

June 13, 2026

Feature GeoLite2 → GeoIP2 Upgrade Path

Pro, Business, and Enterprise plans now use paid MaxMind GeoIP2 databases (95%+ city fill rate) when available, with automatic fallback to GeoLite2. Free and Starter plans remain on GeoLite2.

June 12, 2026

Improve Pro Plan Quota Increase

Increased Pro plan monthly quota from 250,000 to 500,000 lookups. Price remains $29/mo. Added 7-day money-back guarantee to all paid plans.

June 10, 2026

Feature Batch Lookup Endpoint

Added POST /v1/ip/batch — look up up to 100 IPs in a single request. Parallel mmap reads via thread pool executor.

June 8, 2026

Feature Paddle Billing Integration

Integrated Paddle as merchant-of-record for paid plans. Automatic plan provisioning via webhooks. Free plan stays direct-registration with no checkout required.

June 5, 2026

Feature Public Beta Launch

IPGeo launched with MaxMind GeoLite2 data, mmap-backed lookups, Redis billing, and Docker Compose deployment. Free tier: 10,000 queries/month.
